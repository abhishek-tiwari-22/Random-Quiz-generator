# Random-Quiz-generator
Tkinter as GUI in python to create a quiz application taking randon questions from spreadsheet.

---

Random-Quiz-Generator:
A random 10 questions are chosen from the online google spreadsheet (Implemented using SQLite as well) which can contain even 1000 questions.
It was created for college branch which was in demand for some students for teaching purpose during lockdown as everythong goeas offline.
Good thing is every student might not get the same questions for the quiz which will help teachers for the analysis of the students and also they will get their scores immediately.
